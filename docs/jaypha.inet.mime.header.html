enum MimeEoln

End of line sequence. In MIME and IMF, end of lines are always represented by CRLF (carriage return, line feed), or "\r\n".

MimeHeader mimeVersion

A MIME header for representing the MIME version. It is always 1.0.

struct MimeHeader

A struct for MIME/IMF headers.

Rather than implementing a sophisticated polymorphic solution, I opted for a simpler struct based one. Field bodies are expected to be formatted as the header is created. There are many functions and structs that do this.

Properties:

string nameHeader name
string fieldBodyHeader field body
string asStringThe whole header as a string

Example:

  auto header = MimeHeader("X-Stuff", "The content");
  assert(header.asString == "X-Stuff: The content");
