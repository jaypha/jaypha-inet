jaypha.inet.mime.types

This module associates typical file extensions with mime-types. All extensions are given in lower case.

By default, this module will define only the most common ones used. To use a more extensive list, compile your application with the "allMimeTypes" version set.

The list of extensions and their mime type mappings were obtained from http://www.stdicon.com/mimetypes

Warning: There is no formal standard for file extensions, and many extensions are 'claimed' by multiple vendors. There is also no rule enforcing the use of 'correct' file extrensions.

immutable string[string] fileExtMimeTypes

An associative array mapping file extensions to mime types.

string extMimeType(string ext)

Returns the mime type associated with the extension.

Parameters:

string extThe extension (case insensitive).

Returns:
The mime type of the extension. If one is not found, returns 'application/octet-stream' by default.

string fileMimeType(string filename)

Returns the mime type associated with a filename. This function extracts the extension from the filename (the content following the last '.').

Parameters:

string filenameThe filename (case insensitive).

Returns:
The mime type of the filename. If one is not found, or if no extension is found, returns 'application/octet-stream' by default.
