jaypha.inet.mime.contenttype

A structure for MIME Content-Type headers.

The MIME content type header is defined in RFC2045.

The MIME content type value, usually refered to as the "mime type", is given in the form (<type>/<subtype>).

MIME content type headers can also provide extra information in the form of parameters. The parameters are given in the format of <attribute>=<value>, separated by ";". The value can be in quotes and comments are allowed as per RFC822.

Example

Content-Type: text/plain; charset=us-ascii (Plain text)

The mime type is "text/plain". There is one parameter, "charset" of value "us-ascii".

struct MimeContentType

MIME Content-Type.

Use this for creating/reading/storing information relevant to Content-Type MIME headers.

Use extractMimeContentType to read form input, and toMimeHeader to prepare for output.
